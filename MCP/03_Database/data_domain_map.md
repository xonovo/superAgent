# 数据领域地图

## 核心数据域

1. 人
2. 房屋
3. 小区
4. 楼栋
5. 单元
6. 物业企业
7. 合同
8. 收费
9. 公共收益
10. 维修资金
11. 银行流水
12. 产权登记
13. 投诉与事件
14. 风险预警
15. 审计日志

## 数据主线

```text
产权数据
  -> 房屋数据
    -> 人房关系
      -> 小区画像
        -> 物业企业监管
          -> 收费和公共收益
            -> 风险识别
              -> 审计和监管决策
```

## 数据字典最低要求

每个字段必须登记：

1. 字段名
2. 中文名
3. 数据类型
4. 是否必填
5. 是否唯一
6. 来源系统
7. 业务含义
8. 安全等级
9. 是否参与 AI 特征
10. 是否需要审计
