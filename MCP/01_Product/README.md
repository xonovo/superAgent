# 01 Product

本目录负责产品定义、PRD、用户角色、业务流程和验收口径。

## 核心用户

1. 住建局监管人员。
2. 城市物业公司管理人员。
3. 物业企业。
4. 小区业委会或相关自治组织。
5. 银行和资金监管单位。
6. 审计单位。
7. 业主和房屋权利相关人。

## 必备文档

- `prd_outline.md`：PRD 总纲。
- `user_roles.md`：用户角色和权限边界。
- `business_processes.md`：核心业务流程。
- `acceptance_criteria.md`：产品验收标准。

## 责任角色

- 主责：PM Codex
- 协作：Frontend Codex、Backend Codex、UI Codex、Document Codex、Auditor Codex
