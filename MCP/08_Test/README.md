# 08 Test

本目录负责测试策略、测试用例、验收标准和质量门禁。

## 测试范围

1. 单元测试
2. 接口测试
3. 权限测试
4. 数据迁移测试
5. 外部接口联调测试
6. 对账测试
7. UI 测试
8. 移动端兼容测试
9. AI 输出验证
10. 发布回归测试

## 必备文档

- `test_strategy.md`：测试策略。
- `test_case_standard.md`：测试用例标准。
- `acceptance_checklist.md`：验收清单。
- `regression_plan.md`：回归计划。

## 责任角色

- 主责：Test/Auditor Codex
- 协作：所有执行 Codex
