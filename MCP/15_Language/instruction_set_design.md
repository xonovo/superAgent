# Instruction Set Design

## 1. 设计目标

指令集设计必须让 AI Agent、AOEM 内核和业务系统之间的执行边界清晰、可审计、可回滚。

## 2. 指令必须定义

1. 指令名称。
2. 输入参数。
3. 输出结果。
4. 权限要求。
5. 状态变化。
6. 错误码。
7. 审计日志。
8. 回滚或补偿策略。

## 3. 指令分类

1. 只读查询指令。
2. 文档生成指令。
3. 数据处理指令。
4. AI 推理指令。
5. AOEM 执行指令。
6. 高风险生产指令。

## 4. 高风险指令

涉及生产数据、资金、银行接口、AI 自动决策、AOEM 核心执行的指令必须进入 L6 审计。
