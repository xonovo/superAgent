# Learning Ingestion Policy

## 1. 入库流程

```text
新资料
  -> 来源登记
  -> 摘要
  -> 标签
  -> 适用范围
  -> 不适用范围
  -> 可信等级
  -> 知识库归档
  -> 专家训练状态更新
```

## 2. 必填字段

1. 资料名称。
2. 来源。
3. 作者或负责人。
4. 时间。
5. 摘要。
6. 标签。
7. 适用范围。
8. 不适用范围。
9. 关联角色。
10. 可信等级。

## 3. 禁止事项

1. 未登记来源的资料不得进入知识库。
2. 未摘要的资料不得作为任务依据。
3. 未复核的资料不得作为生产依据。
