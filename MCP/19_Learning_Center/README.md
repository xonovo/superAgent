# 19_Learning_Center

本目录归属扁鹊（CLO-001 / Chief Learning Officer），用于管理 OMCF 的知识吸收、蒸馏、训练和专家成长。

## 职责范围

1. 新文档吸收。
2. 知识蒸馏。
3. 训练集整理。
4. 专家成长记录。
5. 专家训练状态登记。

## 核心文档

| 文档 | 说明 |
|---|---|
| `learning_ingestion_policy.md` | 新知识吸收规则 |
| `knowledge_distillation_policy.md` | 知识蒸馏规则 |
| `expert_training_register.md` | 专家训练状态登记 |
| `training_dataset_index.md` | 训练资料索引 |

## 强制规则

1. 新资料不得直接进入专家结论，必须先完成吸收。
2. 专家训练状态未达到 `TRAINED` 不得输出定论。
3. 训练资料必须保留来源、版本、摘要和适用范围。
