# Math Model Validation

## 1. 验证目标

数学模型验证用于确认模型假设、推导过程、误差边界和工程落地条件是否可靠。

## 2. 验证清单

1. 符号定义完整。
2. 假设条件明确。
3. 推导过程可复查。
4. 输入数据口径明确。
5. 输出解释清晰。
6. 误差边界可验证。
7. 反例或失败条件已列出。
8. 工程约束已说明。

## 3. 二次确认

以下模型必须二次确认：

1. 影响 AI 训练或推理结果。
2. 影响资金风险判断。
3. 影响行政审批或监管判断。
4. 引入量子计算、索尔算法或复杂优化假设。
5. 无法通过常规测试完整覆盖。

## 4. 验证证据

验证报告必须归档：

1. 输入样本。
2. 计算过程。
3. 结果摘要。
4. 异常样本。
5. 审计结论。
