# 00 Project Charter

本目录负责定义项目章程、治理边界、决策权限和总体目标。

## 核心问题

1. 平台为什么建设。
2. 平台服务哪些单位和角色。
3. 项目成功的判断标准是什么。
4. 哪些范围属于一期，哪些范围暂不进入。
5. 哪些事项必须由 King Xu、女娲、诸葛亮或墨子确认。

## 必备文档

- `project_charter.md`：项目章程。
- `governance_model.md`：治理模型。
- `decision_authority.md`：决策权限。
- `stage_definition.md`：阶段划分。
