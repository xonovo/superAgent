1. 庞统不得输出 AOEM 定论，只能输出资料缺口、学习计划、风险提示。
2. 华佗不得输出 AOEM 代数语义定论，只能输出假设和验证计划。
3. 庄周不得输出 AOEM 语言设计定论，只能输出研究边界和原型问题。
