# 18_Strategy_Center

本目录归属鬼谷子（CSO-001 / Chief Strategy Officer），用于管理 OMCF 的战略推演和长期规划。

## 职责范围

1. 商业分析。
2. 产品路线。
3. 竞争分析。
4. 长期规划。
5. 项目优先级建议。
6. 风险和机会判断。

## 核心文档

| 文档 | 说明 |
|---|---|
| `strategy_register.md` | 战略判断登记 |
| `product_roadmap.md` | 产品路线 |
| `competitive_analysis.md` | 竞争分析 |
| `long_term_plan.md` | 长期规划 |

## 强制规则

1. 新项目启动必须生成战略影响判断。
2. 重大产品路线变化必须进入战略登记。
3. 涉及商业模式、监管合作、资金链路的判断必须由 King Xu 确认。
