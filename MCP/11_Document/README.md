# 11 Document

本目录负责文档体系、领导汇报材料、招投标材料、PRD、接口文档、测试文档和运维文档。

## Document Codex 职责

1. 建立文档编号规则。
2. 建立文档目录和索引。
3. 维护 PRD、设计文档、数据库文档、接口文档、测试文档和运维文档。
4. 输出领导汇报材料。
5. 输出招投标材料。
6. 对文档版本一致性进行检查。
7. 将审计通过的文档归档。

## 必备文档

- `document_standard.md`：文档标准。
- `reporting_materials.md`：领导汇报材料框架。
- `bidding_materials.md`：招投标材料框架。
- `archive_policy.md`：文档归档规则。

## 责任角色

- 主责：Document Codex
- 协作：所有 Codex
