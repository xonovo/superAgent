# 05 UI

本目录负责 UI 规范、设计系统、页面标准和交互规则。

## UI 目标

本平台是监管和运营工具，界面应安静、清晰、稳定，优先支持高频查询、对比、审核、追踪和批量处理。

## 必备文档

- `design_system.md`：设计系统。
- `page_standards.md`：页面标准。
- `component_rules.md`：组件规则。
- `dashboard_rules.md`：驾驶舱规则。

## 责任角色

- 主责：UI Codex
- 协作：Frontend Codex、Mobile Codex、PM Codex、Document Codex
