# 17_Memory_Center

本目录归属仓颉（CMO-001 / Chief Memory Officer），用于管理 OMCF 的长期记忆。

## 记忆范围

1. 所有项目历史。
2. 所有决策记录。
3. 所有架构记录。
4. 所有审计失败。
5. 所有返工原因。
6. 所有发布记录。
7. 所有失败经验。

## 核心文档

| 文档 | 说明 |
|---|---|
| `decision_log.md` | 决策记录 |
| `architecture_decision_record.md` | 架构决策记录 |
| `task_history.md` | 任务历史 |
| `failure_lessons.md` | 失败经验 |
| `release_memory.md` | 发布记忆 |

## 强制规则

1. 没有记忆检索的任务不得进入 `READY`。
2. 架构变更必须写入 ADR。
3. 审计失败必须写入失败经验。
4. 发布完成必须写入发布记忆。
