# 12 Data Governance

本目录负责 66 万房屋数据治理、数据清洗、质量检查、重复数据识别、产权同步检查、银行同步检查和 AI 训练数据准备。

## Data Governance Codex 职责

1. 建立房屋数据质量标准。
2. 建立人、房、楼栋、小区、物业企业的数据口径。
3. 识别重复数据、缺失数据和异常数据。
4. 检查产权同步一致性。
5. 检查银行同步一致性。
6. 为 AI Codex 准备训练数据和特征数据。
7. 输出数据质量报告和治理进度报告。

## 必备文档

- `data_governance_plan.md`：数据治理计划。
- `quality_rules.md`：数据质量规则。
- `deduplication_rules.md`：重复数据识别规则。
- `property_sync_rules.md`：产权同步检查规则。
- `bank_sync_rules.md`：银行同步检查规则。
- `training_data_policy.md`：AI 训练数据准备规则。

## 责任角色

- 主责：Data Governance Codex
- 协作：Database Codex、AI Codex、Backend Codex、Auditor Codex
