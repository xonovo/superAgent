# 09 Audit

本目录负责审计规则、审计报告、PASS/FAIL 规则和整改闭环。

## 审计范围

1. 数据审计
2. 代码审计
3. 安全审计
4. AI 输出审计
5. 银行接口审计
6. 发布审计
7. 文档审计

## 必备文档

- `audit_checklist.md`：审计清单。
- `audit_report_template.md`：审计报告模板。
- `security_audit.md`：安全审计规则。
- `ai_output_audit.md`：AI 输出审计规则。

## 责任角色

- 主责：Auditor Codex
- 协作：Document Codex、所有执行 Codex
