# Metrics Center

Metrics Center stores local Runtime and agent metrics generated by Runtime V2.6.

Metric JSON files are local operational records and are ignored by Git:

```text
OMCF_Runtime/runtime/metrics/*.json
```

Runtime V2.6 records:

1. Agent invocation counts.
2. Provider execution counts.
3. Provider success rate.
4. Human approval rate.
5. Audit status.
