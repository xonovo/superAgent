# Runtime Engine

本目录包含 OMCF Runtime V1 的执行引擎。

## 命令

```powershell
python OMCF_Runtime/runtime/omcf_runtime.py start-project --project-name "株洲物业监管平台" --project-code "demo_property" --project-type "政务 / 物业监管 / 数据平台"
```

## 输出

Runtime 会生成：

1. Runtime Context
2. 女娲启动记录
3. 诸葛亮任务树
4. 墨子架构树
5. 嬴政文档树
6. 赵云审计报告
